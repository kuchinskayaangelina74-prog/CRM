# CRM-система для управления клиентами и маркетингом

Простая CRM-система, разработанная на **Django 6.0**, реализующая управление услугами, рекламными кампаниями, лидами и контрактами.

## Основной функционал
- **Аналитическая панель**: Расчет выручки, количества лидов и конверсии рекламных компаний.
- **Управление воронкой**: Перевод потенциального клиента (Лида) в статус активного клиента при заключении контракта.
- **Хранение документов**: Загрузка и хранение файлов договоров.
- **База данных**: Интеграция с **PostgreSQL**.

## Технологии
- Python 3.13
- Django 6.0
- PostgreSQL (через порт 5433)
- Bootstrap 5

## Установка и запуск

1. **Клонируйте проект и настройте виртуально окружение:**
```bash
    python -m venv venv
    source venv/bin/activate  
    pip install -r requirements.txt
```

2. **Настройка базы данных (PostgreSQL):**
    - Убедитесь, что PostgreSQL запущен на порту `5433`.
    - Создайте базу данных `crm_db`.

3. **Запуск проекта:**
```bash
   python manage.py migrate
   python manage.py createsuperuser
   python manage.py runserver
```

4. **Доступ к системе:**
   - Главная страница: `http://127.0.0.1:8000/`
   - Админ-панель: `http://127.0.0.1:8000/admin/`
